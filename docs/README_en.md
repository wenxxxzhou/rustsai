## _prop\edl2srt

- This is a Rust program that converts PR EDL files into standard SRT files;
- It may also be compatible with EDL files generated by other programs;
- Build it using `cargo build --release`;
- View the instructions via `.\target\release\edl2srt.exe --help`;
  - Or use `cargo run --release -- --help`;
- Run it with `.\target\release\edl2srt.exe <arguments>`;
