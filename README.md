## _prop\edl2srt

- 这是一个使 PR 的 EDL 转换为标准 SRT 的 RUST 程序；
- 或许还兼容其他程序输出的 EDL 文件；
- 构建 `cargo build --release` ；
- 查看说明 `.\target\release\edl2srt.exe --help` ；
  - 或 `cargo run --release -- --help` ;
- 运行 `.\target\release\edl2srt.exe <参数>`；

## _prop\zqinfoss

- 一个用于解析 Steam 游戏链接，自动获取游戏名并生成 SteamDB 与 HLTB 查询链接的命令行工具，内置本地缓存以优化查询速度
- 构建
- 或下载 EXE 后双击运行
  - 简单开始：单击 steam 客户端左上链接进行复制，直接粘贴进 CLI 中并 enter 开始查询
