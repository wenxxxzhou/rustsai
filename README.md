## _prop\edl2srt

- 这是一个使 PR 的 EDL 转换为标准 SRT 的 RUST 程序；
- 或许还兼容其他程序输出的 EDL 文件；
- 构建 `cargo build --release` ；
- 查看说明 `.\target\release\edl2srt.exe --help` ；
  - 或 `cargo run --release -- --help` ;
- 运行 `.\target\release\edl2srt.exe <参数>`；
